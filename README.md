# Highcharts template

グラフ描画ライブラリHighcharts用のテンプレート

## ディレクトリ構造
```bash
.
├── src
│   ├── highcharts
│   │   ├── highcharts.js
│   │   └── ...
│   ├── data.json
│   └── index.js
├── deno.json
├── deno.lock
├── main.tsx
└── README.md
```

## 実行方法
1. `deno task start`を実行
2. http://localhost:8000 を開く
